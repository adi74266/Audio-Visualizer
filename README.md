# Audio-Visualizer
A simple Audio-Visualizer circuit

## Components
1. I2S INMP441 Omnidirectional Microphone Module
  An omnidirectional microphone which unlike other microphones, reads analog data and sends digital data for processing. 
2. I2C SSD1306 0.96" OLED Display
   An OLED screen used for displaying contents

### Pinout
For Microphone:
W2S pin     ->   5
L/R pin     ->  GND
SDK pin     ->   4
SD pin      ->   6

For SSD1306:
SCK pin     ->   12
SDA pin     ->   11
